EMS React + JWT (minimal)
------------------------

خطوات التشغيل المحلية:

1) تثبيت الاعتمادات:
   npm install

2) تشغيل السيرفر التجريبي (من مجلد المشروع):
   node server/index.js

3) تشغيل الواجهة (بافتراض Vite مثبت):
   npm run dev

ملاحظات أمنية:
- لا تخزن توكنات حساسة في localStorage للبيئات الإنتاجية بدون تدابير أمان.
- استبدل SECRET بسلسلة قوية واحفظها في متغيرات البيئة.
