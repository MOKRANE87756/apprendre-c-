ما هي لغة C++؟

C++ هي لغة برمجة متعددة الاستخدامات تم تطويرها بواسطة بيورن ستروستروب. تُستخدم لتطوير البرمجيات عالية الأداء مثل الألعاب، أنظمة التشغيل، والتطبيقات العلمية.

أمثلة عملية

مثال 1: حساب مجموع عددين


#include 

int main() {
    int a = 5, b = 7;
    int sum = a + b;
    std::cout << "المجموع: " << sum << std::endl;
    return 0;
}

مثال 2: طباعة جدول الضرب


#include 

int main() {
    for (int i = 1; i <= 10; i++) {
        std::cout << "5 x " << i << " = " << 5 * i << std::endl;
    }
    return 0;
}

اختبار سريع

ما هو الناتج عند تشغيل البرنامج التالي؟


#include 

int main() {
    int x = 5;
    int y = 10;
    std::cout << x + y;
    return 0;
}
